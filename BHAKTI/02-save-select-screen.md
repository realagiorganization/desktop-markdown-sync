# Bhakti 02 - Save Select Screen

- Screen type: `profile and save management`
- Platform framing: `dual-screen save menu`
- Functional purpose: Choose a save file, review progress, or create a new profile.
- Visible content summary: The top screen shows three stylized save cards with shrine thumbnails, playtime, and completion symbols. The bottom screen contains touch targets for copy, erase, and confirm actions.
- Interface details: Each slot lists verses learned, murtis unlocked, and translation notes collected. A stylus tap opens a compact preview of the player’s most recent study session.
- Device presence: The lower touchscreen uses large tap zones typical of Nintendo DS menu navigation.

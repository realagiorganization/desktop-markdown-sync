# Bhakti 03 - Monastery Courtyard Map

- Screen type: `exploration hub`
- Platform framing: `top screen world view, bottom screen map controls`
- Functional purpose: Move between shrines, study halls, libraries, and festival areas.
- Visible content summary: The top screen shows a compact monastery courtyard with stone paths, lanterns, gardens, and small prayer pavilions. The bottom screen displays a touch map with icons for available activities.
- Interface details: The player taps locations to move, while contextual callouts show which texts, teachers, or interactive murtis are available in each area.
- Device presence: The lower screen works as the main navigation surface with stylus-driven fast travel.

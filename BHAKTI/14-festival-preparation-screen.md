# Bhakti 14 - Festival Preparation Screen

- Screen type: `event setup`
- Platform framing: `dual-screen timed activity`
- Functional purpose: Prepare decorations, select verses, and arrange ritual order before a festival scene.
- Visible content summary: The top screen previews the decorated courtyard or hall as arrangements are made. The bottom screen contains inventory strips for lamps, garlands, banners, and selected chant sets.
- Interface details: The player drags decorations into place and assigns text recitations to event stages. Correct combinations can unlock rare transcripts or special archive scenes.
- Device presence: The touchscreen manages item placement and event sequencing.

# Bhakti 17 - Pilgrimage Route Screen

- Screen type: `progression map`
- Platform framing: `top screen illustrated route, bottom screen path planner`
- Functional purpose: Choose the next pilgrimage destination and review requirements for travel-linked study activities.
- Visible content summary: The top screen shows a route across forests, rivers, village gates, and shrine complexes. The bottom screen marks available stops, collectible texts, and practice sessions tied to each segment.
- Interface details: Players tap nodes to inspect rewards, difficulty, and which translations or chant sets can be discovered. Some paths branch based on completed lessons or archive progress.
- Device presence: Route selection is handled through bottom-screen node taps.

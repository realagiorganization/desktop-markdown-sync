# Bhakti Screen Set

This directory collects text-only descriptions of functional game screens for a fictional Nintendo DS game focused on interactive murtis, touchscreen study flows, and overlays with transcriptions and translations of Vedic texts.

- Scope: dual-screen gameplay, stylus interaction, text-study overlays, shrine navigation, chanting tools, and progress or archive screens tied to active play.
- Format: one Markdown file per functional screen with concise notes describing what the player sees and what actions the screen supports.
- Intent: repo-friendly screenshot descriptions that stay text-first and avoid binary capture artifacts.

# Console Game 05 - The Legend of Zelda: Ocarina of Time

- Generation: `Nintendo 64 / fifth generation`
- Original platform: `Nintendo 64`
- Release era: `1998`
- Fan-favorite reason: Lock-on combat, cinematic dungeon framing, and an unusually coherent jump to 3D made it a landmark console release.
- Visible content summary: The fixture focuses on early 3D camera language, ocarina-triggered world-state changes, and dungeons built around readable traversal verbs.

## Debug IDE Ideas

- `Project64` with debugger builds or `mupen64plus` tools for CPU stepping, memory watches, and display-list investigation.
- `Visual Studio Code` for decomp project navigation, symbol search, and scripted asset extraction workflows.
- `Ghidra` for binary analysis when comparing ROM revisions or inspecting engine subsystems outside the decomp tree.

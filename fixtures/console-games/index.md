# Console Games Fixture Set

This subdirectory collects text-only fixture notes for 20 fan-favorite console games across several hardware generations.

- Scope: NES through Nintendo Switch, PlayStation 5, and Xbox 360 era favorites.
- Format: one markdown file per game with concise notes plus debugger and IDE ideas for emulation, modding, tooling, or reverse-engineering workflows.
- Intent: repo-friendly sample content that stays text-first and avoids binary capture artifacts.

# Bhakti 05 - Murti Rotation View

- Screen type: `object inspection`
- Platform framing: `upper display model view, lower display controls`
- Functional purpose: Rotate an interactive murti, inspect details, and reveal hidden lore markers.
- Visible content summary: The top screen shows a detailed murti model against a dim shrine backdrop, with highlights on ornaments, hands, and symbolic items. The bottom screen contains a circular stylus control pad and zoom buttons.

- Device presence: The touchscreen is dedicated to rotation and zoom gestures.

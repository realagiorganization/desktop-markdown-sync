# Bhakti 18 - Collection Archive Screen

- Screen type: `completion archive`
- Platform framing: `dual-screen records menu`
- Functional purpose: Review unlocked murtis, texts, transcripts, translations, and audio entries.
- Visible content summary: The top screen highlights the currently selected collection item with artwork or a verse excerpt. The bottom screen lists categories and completion percentages in a compact scrollable layout.
- Interface details: Filters let the player sort by text family, shrine origin, translation status, or audio availability. Selecting an item can jump directly into playback or manuscript inspection.
- Device presence: Archive browsing depends on stylus scrolling and tap targets on the lower screen.

# Bhakti 12 - Library Shelf Browser

- Screen type: `archive navigation`
- Platform framing: `top screen shelf view, bottom screen category tabs`
- Functional purpose: Browse available texts, commentaries, and unlocked study materials.
- Visible content summary: The top screen depicts a warm wooden shelf with scrolls, bound manuscripts, and labeled drawers. The bottom screen contains tabs for hymns, dialogues, rituals, glossaries, and notes.
- Interface details: Tapping an item reveals a short summary, completion status, and whether it includes translation layers, chant playback, or annotation unlocks.
- Device presence: The lower screen is the browsing surface, optimized for quick stylus selection.

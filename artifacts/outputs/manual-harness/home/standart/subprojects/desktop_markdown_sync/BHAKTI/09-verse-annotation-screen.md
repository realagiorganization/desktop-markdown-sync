# Bhakti 09 - Verse Annotation Screen

- Screen type: `study notebook`
- Platform framing: `top screen verse, bottom screen notes`
- Functional purpose: Add annotations, bookmark phrases, and store personal study notes.
- Visible content summary: The top screen keeps the selected verse visible in a clean parchment-style frame. The bottom screen contains lined note space, colored markers, and touch buttons for symbols or tags.
- Interface details: The player can attach notes to individual lines, apply topic tags, and mark phrases for review. Predefined icons help compensate for the DS’s compact handwriting area.
- Device presence: The bottom touchscreen functions as a small interactive notebook.

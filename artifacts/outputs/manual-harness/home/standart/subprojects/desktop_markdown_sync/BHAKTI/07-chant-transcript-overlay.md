# Bhakti 07 - Chant Transcript Overlay

- Screen type: `text and audio study`
- Platform framing: `top screen chant scene, bottom screen transcript overlay`
- Functional purpose: Follow a chant line by line with synchronized transcription.
- Visible content summary: The top screen shows a calm ritual scene with lamp light and subtle movement. The bottom screen displays a scrolling transcript panel that highlights each line as the audio advances.
- Interface details: Controls allow pause, replay, line repeat, and slower playback. A toggle swaps between transliteration and original-script presentation without leaving the chant.
- Device presence: The DS bottom screen acts as the dedicated transcript reader and playback control surface.

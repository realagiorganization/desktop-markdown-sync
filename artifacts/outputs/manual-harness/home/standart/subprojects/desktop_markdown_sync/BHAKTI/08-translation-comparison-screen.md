# Bhakti 08 - Translation Comparison Screen

- Screen type: `parallel text study`
- Platform framing: `dual-screen comparison mode`
- Functional purpose: Compare multiple translations of a selected Vedic verse.
- Visible content summary: The top screen shows the source text with verse numbering and key phrase markers. The bottom screen splits into two or three selectable translation panes with stylus tabs.
- Interface details: Tapping a phrase on the lower screen highlights the corresponding source segment above. The player can pin favorite interpretations or switch commentary layers for a more literal or poetic reading.
- Device presence: Translation selection and pane switching are handled through touchscreen tabs.

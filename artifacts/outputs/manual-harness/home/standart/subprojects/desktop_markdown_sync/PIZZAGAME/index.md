# PizzaGame Screen Set

This directory collects text-only descriptions of functional game screens for a fictional game about a scooter delivery worker in Pattaya, Thailand who uses mobile devices to probe terminals, bypass security screens, and complete pizza jobs under pressure.

- Scope: delivery routes, scooter HUDs, mobile hacking flows, terminal overlays, and support screens tied to active gameplay.
- Format: one Markdown file per functional screen with concise notes describing what the player sees and what actions the screen supports.
- Intent: repo-friendly screenshot descriptions that stay text-first and avoid binary capture artifacts.

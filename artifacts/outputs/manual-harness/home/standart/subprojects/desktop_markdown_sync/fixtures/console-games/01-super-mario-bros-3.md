# Console Game 01 - Super Mario Bros. 3

- Generation: `NES / third generation`
- Original platform: `Nintendo Entertainment System`
- Release era: `1988-1990`
- Fan-favorite reason: Precision platforming, memorable world themes, and the raccoon and tanooki power-up loop made it one of the defining 8-bit console games.
- Visible content summary: The fixture centers on bright side-scrolling stages, dense secret placement, world-map progression, and fast-feedback movement design.

## Debug IDE Ideas

- `Mesen 2` debugger for CPU, PPU, memory, and event tracing when investigating sprite timing or level scripting.
- `Visual Studio Code` with ca65 or NESASM syntax support for ROM hacks, build tasks, and disassembly notes.
- `FCEUX` hex and tile viewers for quick inspection of palette, tile, and map data when debugging assets.

# Console Game 08 - Halo: Combat Evolved

- Generation: `Xbox / sixth generation`
- Original platform: `Microsoft Xbox`
- Release era: `2001`
- Fan-favorite reason: Sandbox combat, readable AI roles, and wide-open shooter encounters made it the defining original Xbox release for many players.
- Visible content summary: This fixture tracks two-weapon combat, vehicle transitions, shield-based encounter pacing, and mission spaces designed for replayable tactical improvisation.

## Debug IDE Ideas

- `Xemu` for Xbox emulation with serial logs and hardware-faithful testing loops.
- `Visual Studio Code` for tag-file editing workflows, map annotations, and community toolchain scripting.
- `Ghidra` or `IDA` for native executable analysis when investigating engine behavior or restoration patches.

# Console Game 11 - Resident Evil 4

- Generation: `GameCube / sixth generation`
- Original platform: `Nintendo GameCube`
- Release era: `2005`
- Fan-favorite reason: Its over-the-shoulder combat camera and encounter pacing redefined action-horror on consoles and influenced a generation of third-person shooters.
- Visible content summary: The main cues are laser-aim precision, pressure-heavy crowd encounters, and level scripting that keeps escalating without losing readability.

## Debug IDE Ideas

- `Dolphin Emulator` for event breakpointing, animation-state observation, and memory inspection on the GameCube original.
- `Visual Studio Code` for mod loader configs, data-file notes, and binary patch automation.
- `Ghidra` for executable comparison across console and later remaster variants when hunting logic changes.

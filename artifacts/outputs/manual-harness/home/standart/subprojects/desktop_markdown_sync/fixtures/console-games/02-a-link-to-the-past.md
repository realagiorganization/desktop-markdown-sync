# Console Game 02 - The Legend of Zelda: A Link to the Past

- Generation: `SNES / fourth generation`
- Original platform: `Super Nintendo Entertainment System`
- Release era: `1991-1992`
- Fan-favorite reason: Its dual-world structure, clean dungeon design, and near-perfect action-adventure pacing set the template for console Zelda for years.
- Visible content summary: This fixture emphasizes overworld routing, mirrored Light and Dark World logic, item-gated progression, and compact but expressive top-down combat.

## Debug IDE Ideas

- `bsnes-plus` debugger for SNES tracing, VRAM inspection, and breakpoint-driven analysis of map or item logic.
- `Visual Studio Code` for annotated 65816 disassembly, patch manifests, and reproducible ROM-hack build scripts.
- `Geiger's SNES9x Debugger` for quick RAM watches when exploring room state, flags, and transition behavior.

# Console Game 07 - Tony Hawk's Pro Skater 2

- Generation: `PlayStation / fifth generation`
- Original platform: `Sony PlayStation`
- Release era: `2000`
- Fan-favorite reason: The combo system, soundtrack, and immediate mastery curve turned score-chasing into a near-perfect pick-up-and-play console loop.
- Visible content summary: The relevant signals are compact arenas, high-speed route linking, multiplier preservation, and objective design that pushes experimentation without stopping momentum.

## Debug IDE Ideas

- `DuckStation` or `PCSX-Redux` for frame stepping and memory watches around scoring and combo state.
- `Visual Studio Code` for level-logic notes, texture replacement manifests, and scripting helpers used by modding communities.
- `Noesis` plus hex tooling when inspecting archived assets and model formats extracted from disc images.

# Console Game 03 - Super Metroid

- Generation: `SNES / fourth generation`
- Original platform: `Super Nintendo Entertainment System`
- Release era: `1994`
- Fan-favorite reason: Atmosphere, sequence-breaking depth, and tightly interconnected exploration keep it central to speedrunning and game-design discussions.
- Visible content summary: The key signals are lonely sci-fi spaces, layered map interconnections, controlled mobility unlocks, and combat systems that reward route knowledge.

## Debug IDE Ideas

- `Mesen-S` or `bsnes-plus` for breakpointing room loads, enemy AI behavior, and DMA-heavy transitions.
- `Visual Studio Code` with asm highlighting and task runners for IPS or BPS patch workflows used by randomizer and hack authors.
- `SMILE RF` or related room editors when inspecting level data alongside emulator traces.

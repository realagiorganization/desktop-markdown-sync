# Shipgirls 02 - Monastery Fleet Map

- Screen type: `mission selection`
- Location framing: `Buddhist-Bhakti monastery confederation theater map`
- Functional purpose: Select operations, inspect routes, and assign carrier, aircraft, or spacecraft deployments to contested zones.
- Visible content summary: The strategic map shows island monasteries, orbital docks, pilgrimage corridors, and data shrines linked by shipping lanes and launch vectors. Mission markers indicate logistics work, defensive patrols, and terminal-seizure objectives.
- Interface details: Each mission card lists fleet composition, weather or vacuum conditions, signal interference, and required hacking specialties. Some routes offer safer travel while others pass through dense robotic surveillance networks with better rewards.

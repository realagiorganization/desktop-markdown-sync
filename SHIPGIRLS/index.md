# Shipgirls Screen Set

This directory collects text-only descriptions of functional game screens for a fictional vehicle game set in a sci-fi Buddha Japan Edo-punk world. The game centers on aircraft carriers, aircraft, and spacecraft systems while the player hacks terminals connected to robotic computing networks associated with anime-styled shipgirls who serve within a Buddhist-Bhakti monastery confederation.

- Scope: carrier bridges, flight decks, orbital craft controls, terminal intrusion screens, support systems, and mission management interfaces tied to active gameplay.
- Format: one Markdown file per functional screen with concise notes describing visible elements and player actions.
- Intent: repo-friendly screenshot descriptions that stay text-first and avoid binary capture artifacts.
